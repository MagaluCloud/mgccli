# Flavors available for database instances.

## Usage:
```bash
Usage:
  ./mgc dbaas flavors [flags]
  ./mgc dbaas flavors [command]
```

## Product catalog:
- Commands:
- get         Flavor detail.
- list        List available flavors.

## Other commands:
- Flags:
- -h, --help      help for flavors
- -v, --version   version for flavors
