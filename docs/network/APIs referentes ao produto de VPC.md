# APIs referentes ao produto de VPC

## Usage:
```bash
Usage:
  ./mgc network [flags]
  ./mgc network [command]
```

## Product catalog:
- Commands:
- ports           VPC Port
- public-ips      VPC Public IPs
- rules           VPC Rules
- security-groups VPC Security Groups
- subnets         VPC Subnets
